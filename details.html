Overview

This application provides the user with a virtual scoreboard. It allows them to record the number of points for their selected ‘players’. Each players score is recorded individually, and all are displayed at the same time displayed. The scores can be increased or decreased by whichever increment to user chooses. The minimum and maximum for each score has been set at ‘0’ and ‘999’ respectively. A total of 10 players can be entered at any one time.
This project was built primarily with jQuery. The ‘.append()’ method is used is generate the input field for entering the name of each player. A text input field and corresponding label element are created for each individual player. The default number of players is set at ‘1’, however this can be increases by the user by clicking ‘add player’. This runs a function which increases the variable representing the number of players and implement the ‘.append()’ method to generate another input field and label.
When the player clicks ‘start’, the ‘.hide()’ method is used to get rid of the set-up menu. The ‘.each()’ method is used on an array containing the player name input fields to generate a scoreboard for each player. For each player, the ‘createElement()’ method is used to create a div with the class of ‘board’. The ‘.append()’ method is used to add elements into the ‘board’ div. These elements display the players name, their score, which starts at ‘0’, and an input field with accompanying buttons for adjusting their score. ‘.on(“click”)’ events are added to each button. These events run functions which calculate and update the players score.
Before a scoreboard is generated, an ‘if’ statement is used to check if the player name has been left blank and if so then a scoreboard is not generated. ‘if’ statements are also used to set the lower and upper limits for the players score and changing the font-size depending on its length.
